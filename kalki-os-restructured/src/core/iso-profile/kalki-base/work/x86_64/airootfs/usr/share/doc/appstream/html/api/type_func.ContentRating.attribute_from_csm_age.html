Description [src]

Gets the highest AsContentRatingValue which is allowed to be seen by the given Common Sense Media age for the given subsection id.

For example, if the CSM age mappings for violence-bloodshed are: * age ≥ 0 for AS_CONTENT_RATING_VALUE_NONE * age ≥ 9 for AS_CONTENT_RATING_VALUE_MILD * age ≥ 11 for AS_CONTENT_RATING_VALUE_MODERATE * age ≥ 18 for AS_CONTENT_RATING_VALUE_INTENSE then calling this function with violence-bloodshed and age set to 17 would return AS_CONTENT_RATING_VALUE_MODERATE. Calling it with age 18 would return AS_CONTENT_RATING_VALUE_INTENSE.
